CyberShield IDS/IPS v2.0

Hệ thống phát hiện và ngăn chặn xâm nhập mạng tiên tiến được xây dựng bằng Python, Flask và các công nghệ hiện đại. Hỗ trợ giám sát thời gian thực, phân tích giao thức và tự động ngăn chặn các mối đe dọa.

IDS - Intrusion Detection System

Hệ thống Phát hiện xâm nhập hoạt động bằng cách giám sát lưu lượng mạng và ghi nhận các hoạt động đáng ngờ. IDS không can thiệp vào luồng dữ liệu mà chỉ cảnh báo cho quản trị viên.

IPS - Intrusion Prevention System

Hệ thống Ngăn chặn xâm nhập mở rộng tính năng của IDS bằng cách chủ động can thiệp vào luồng dữ liệu - tự động chặn traffic độc hại trước khi nó tiếp cận hệ thống đích.

Ngăn chặn thời gian thực
Các loại tấn công được phát hiện
DoS / DDoS
SYN Flood, UDP Flood, ICMP Flood, HTTP Flood, DNS Amplification. Tấn công từ chối dịch vụ làm quá tải băng thông hoặc tài nguyên hệ thống.
Web Attacks
SQL Injection, XSS (Cross-Site Scripting), Directory Traversal, Command Injection. Tấn công nhắm vào các ứng dụng web.
Reconnaissance
Port Scan, Nmap OS Detection, Network Mapping. Giai đoạn thu thập thông tin trước khi tấn công thực sự.
Brute Force
SSH Brute Force, FTP Brute Force, RDP Brute Force. Thử nhiều mật khẩu để đăng nhập trái phép.
Man-in-the-Middle
ARP Spoofing, DNS Spoofing. Kẻ tấn công chèn mình vào giữa các kết nối mạng.
Malware Traffic
C&C Communication, Botnet Traffic. Phát hiện lưu lượng kết nối đến máy chủ điều khiển phần mềm độc hại.
